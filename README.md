## Features

**User Management**

- User registration and secure login
- Role-based access control (Admin, Staff)
- Admin can add, update, or remove staff accounts

**Shift Management**

- Create, update, and delete shifts
- Assign shifts directly to staff
- Staff can view their assigned shifts in real time

**Shift Requests**

- Staff can request new shifts or swap existing ones
- Admin can approve or decline requests
- System automatically updates shift assignments after approval

**Notifications**

- Alerts for newly published rosters
- Notifications for approved or declined shift requests
- Real-time updates to keep staff informed
